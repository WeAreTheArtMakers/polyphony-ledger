# Platform Genel Bakis

Polyphony Ledger, su katmanlari tek yapida birlestiren gercek zamanli kripto odeme platformudur:

- immutable double-entry ledger dogrulugu
- replay edilebilir operasyonel projection
- streaming-native event isleme
- dusuk gecikmeli OLAP analitik
- uctan uca gozlemlenebilirlik

## Ana Deger

Platform, muhasebe dogrulugu ile analitik hiz arasindaki klasik tavizi ortadan kaldirmayi hedefler.
