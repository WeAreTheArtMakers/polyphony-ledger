# Urun Paketleri

## Paket A: Foundation

Erken asama teknik dogrulama icin uygundur.

Icerik:

- temel pipeline (ingest -> ledger -> projection -> analytics)
- baseline observability
- schema governance ve replay destegi

## Paket B: Scale

Kontrollu canli trafik pilotu icin uygundur.

Foundation +

- gelismis alerting
- governance endpointleri (quota + usage)
- smoke/e2e kalite kapilari

## Paket C: Enterprise

Guvenlik ve uyumluluk odakli ortamlara uygundur.

Scale +

- SSO entegrasyon yolu
- guclu RBAC politikalari
- ileri seviye operasyonel kontrol ve runbook

## Opsiyonel Eklentiler

- managed operations
- ozel dashboard ve KPI setleri
- uyumluluk odakli retention/audit export
