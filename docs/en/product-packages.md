# Product Packages

## Package A: Foundation

Best for early-stage teams validating architecture.

Includes:

- core pipeline (ingest -> ledger -> projection -> analytics)
- baseline observability
- schema governance and replay support

## Package B: Scale

Best for production pilot with controlled business traffic.

Includes everything in Foundation, plus:

- enhanced alerting
- governance endpoints (quota + usage)
- delivery gates with smoke/e2e verification

## Package C: Enterprise

Best for security and compliance-heavy workloads.

Includes everything in Scale, plus:

- SSO integration path
- stronger RBAC policies
- advanced operational controls and runbooks

## Optional Add-ons

- managed operations
- custom dashboards and KPI packs
- compliance-specific data retention and audit exports
